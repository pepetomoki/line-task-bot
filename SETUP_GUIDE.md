# 🔰 初心者向け セットアップガイド

このガイドでは、タスク管理Botを動かすために必要な **3つのサービスの設定** を、
画面の操作を一つずつ説明しながら進めていきます。

---

## 📌 全体の流れ

```
ステップ1: LINE公式アカウントを作る（10分）
ステップ2: Google Gemini API Keyを取得する（3分・無料）
ステップ3: Googleカレンダーの設定をする（10分）
ステップ4: 設定ファイルを書く（3分）
ステップ5: Botを起動してテストする（5分）
```

---

## ステップ1: LINE公式アカウントを作る 📱

### 1-1. LINE Developersにアクセス

1. ブラウザで https://developers.line.biz/ja/ を開く
2. 右上の **「ログイン」** をクリック
3. 普段使っているLINEアカウントでログイン
   - LINEアプリでQRコードを読み取るか、メールアドレスでログイン

### 1-2. プロバイダーを作る

1. ログイン後、**「プロバイダー」** の横にある **「作成」** をクリック
2. プロバイダー名を入力（例: `マイBot`）→ **「作成」**

### 1-3. チャンネルを作る

1. 作成したプロバイダーのページで **「Messaging API」** を選択
2. 以下を入力:
   - **チャンネル名**: `タスク管理Bot`（好きな名前でOK）
   - **チャンネル説明**: `タスク管理用`
   - **大業種・小業種**: 適当に選択
   - **メールアドレス**: あなたのメールアドレス
3. 利用規約に同意 → **「作成」**

### 1-4. 必要な情報を3つメモする

#### ① Channel Secret
1. 作成したチャンネルを開く
2. **「チャンネル基本設定」** タブをクリック
3. **「チャンネルシークレット」** の値をコピー → メモ帳に貼り付け

#### ② Channel Access Token
1. **「Messaging API設定」** タブをクリック
2. 一番下の **「チャンネルアクセストークン（長期）」** → **「発行」** をクリック
3. 表示された長い文字列をコピー → メモ帳に貼り付け

#### ③ あなたのUser ID
1. **「チャンネル基本設定」** タブに戻る
2. 一番下の **「あなたのユーザーID」** をコピー → メモ帳に貼り付け
   - `U` で始まる英数字の文字列です

### 1-5. 応答設定を変更する

1. **「Messaging API設定」** タブ
2. **「応答メッセージ」** → **「無効」** に変更
3. **「あいさつメッセージ」** → **「無効」** に変更

### 1-6. 友だち追加する

1. **「Messaging API設定」** タブにあるQRコードをスマホのLINEで読み取る
2. 友だちに追加する

> [!TIP]
> これでLINE側はここまで完了！あとで **Webhook URL** も設定しますが、それはBot起動後です。

## ステップ2: Google Gemini API Keyを取得する（無料） 🤖

### 2-1. API Keyを発行

1. https://aistudio.google.com/apikey にアクセス
2. Googleアカウントでログイン
3. **「APIキーを作成」** をクリック
4. プロジェクトを選択（なければ自動作成される）→ **「既存のプロジェクトでAPIキーを作成」**
5. 表示されたキー（`AIza`で始まる文字列）をコピー → メモ帳に貼り付け

> [!TIP]
> Gemini APIは **無料枠** があります（1分間15リクエスト）。個人利用なら十分です！

## ステップ3: Googleカレンダーの設定 📅

> [!NOTE]
> Googleカレンダー連携は後からでも追加できます。まずLINE+AIだけで試したい場合は **このステップをスキップ** して大丈夫です！

### 3-1. Google Cloud Consoleにアクセス

1. https://console.cloud.google.com/ にアクセス
2. Googleアカウントでログイン

### 3-2. プロジェクトを作成

1. 画面上部の **プロジェクト選択** ドロップダウンをクリック
2. **「新しいプロジェクト」** をクリック
3. プロジェクト名: `task-bot` → **「作成」**
4. 作成したプロジェクトを選択

### 3-3. Calendar APIを有効化

1. 左メニュー **「APIとサービス」** → **「ライブラリ」**
2. 検索バーに **「Google Calendar API」** と入力
3. **「Google Calendar API」** をクリック → **「有効にする」**

### 3-4. OAuth認証情報を作成

1. 左メニュー **「APIとサービス」** → **「認証情報」**
2. **「＋認証情報を作成」** → **「OAuthクライアントID」**
3. 初回は **「同意画面を構成」** が必要:
   - **ユーザータイプ**: 「外部」 → **「作成」**
   - アプリ名: `task-bot`、メール記入
   - スコープは追加不要 → **「保存して次へ」**
   - テストユーザーに **自分のGmail** を追加 → **「保存」**
4. 再び **「認証情報」** → **「＋認証情報を作成」** → **「OAuthクライアントID」**
5. アプリケーション種類: **「デスクトップアプリ」** → **「作成」**
6. **「JSONをダウンロード」** をクリック

### 3-5. ファイルを配置

ダウンロードしたJSONファイルの名前を `credentials.json` に変更して、プロジェクトフォルダに移動:

```bash
# ダウンロードフォルダにあるファイルを移動（ファイル名は実際のものに置き換えてください）
mv ~/Downloads/client_secret_xxxx.json /Users/pepetomoki/code/タスク管理/credentials.json
```

## ステップ4: 設定ファイルを書く ✏️

### 4-1. .envファイルを作成

ターミナルで以下を実行:

```bash
cd /Users/pepetomoki/code/タスク管理
cp .env.example .env
```

### 4-2. .envファイルを編集

`.env` ファイルを開いて、メモした値を貼り付けます:

```env
LINE_CHANNEL_SECRET=ステップ1-4①でコピーした値
LINE_CHANNEL_ACCESS_TOKEN=ステップ1-4②でコピーした値
LINE_USER_ID=ステップ1-4③でコピーした値
GEMINI_API_KEY=ステップ2-1でコピーした値
```

> [!TIP]
> `=` の前後にスペースを入れないでください！

## ステップ5: Botを起動する 🚀

### 5-1. サーバーを起動

```bash
cd /Users/pepetomoki/code/タスク管理
python3 app.py
```

以下のように表示されればOK:
```
==================================================
🤖 LINE タスク管理Bot 起動中...
==================================================
🌐 サーバー: http://localhost:5000
📡 Webhook URL: http://localhost:5000/callback
==================================================
```

### 5-2. ngrokでインターネットに公開

**別のターミナルウインドウ** を開いて:

```bash
# ngrokが未インストールの場合:
brew install ngrok

# トンネルを作成
ngrok http 5000
```

表示された `https://xxxx-xxx.ngrok-free.app` のURLをコピー

### 5-3. LINE DevelopersにWebhook URLを設定

1. LINE Developers Console に戻る
2. チャンネル → **「Messaging API設定」** タブ
3. **「Webhook URL」** に以下を入力:
   ```
   https://xxxx-xxx.ngrok-free.app/callback
   ```
   （`/callback` を忘れずに！）
4. **「更新」** → **「検証」** をクリック → 「成功」が出ればOK！
5. **「Webhookの利用」** を **オン** にする

### 5-4. テストしてみよう！ 🎉

1. スマホのLINEを開く
2. 友だち追加したBotアカウントにメッセージを送る:
   ```
   明日の14時にミーティング
   ```
3. AIが解析してタスクを登録してくれるはず！
4. `タスク一覧` と送ると登録されたタスクが見れます
5. `完了 1` と送るとタスクを完了にできます

## 🆘 うまくいかない時は？

| 症状 | 解決方法 |
|---|---|
| `❌ 環境変数が設定されていません` | `.env` ファイルの値が正しくコピーされているか確認 |
| LINEでメッセージを送っても反応なし | ngrokが動いているか確認 / Webhook URLに `/callback` があるか確認 |
| `Webhook検証` が失敗する | `python3 app.py` が動いているか確認 / ngrokのURLが正しいか確認 |
| Google認証でエラー | `credentials.json` がプロジェクトフォルダにあるか確認 |
